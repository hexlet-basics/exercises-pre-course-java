
Метод `Math.random()` возвращает случайное число от 0 до 1 с большим количеством знаков после запятой. Но в реальных задачах бывает нужно получать случайные целые числа. Реализуйте код, который печатает на экран случайное целое число от 0 до 10. Чтобы получить такое число, нужно умножить результат вызова `Math.random()` на 10 и преобразовать тип полученного числа из *double* в *int*.

```java
// Преобразование в int
(int) 0.932342; // 0
(int) 8.123412; // 8
```
